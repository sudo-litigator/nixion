# Changelog

## v0.1.0 - 2026-04-22

- Initial public release of `nixion`
- Added a Rust TUI for browsing installed profile packages and searching `nixpkgs`
- Added flake discovery, flake metadata inspection, and host selection for `nixosConfigurations`
- Added `nix flake update`, `nix flake check`, `nix fmt`, and `nixos-rebuild` actions from the TUI
- Added NixOS system generation browsing, rollback targeting, activation, boot scheduling, deletion, and cleanup preview
